프로젝트 역할분담(현재 진행중)

웹크롤링: 신호철(야놀자 호텔), 김만기(네이버), 김성진(야놀자 모텔)
데이터분석: 정지용(관광데이터 분석), 이가연(교통데이터 분석)
웹개발: 김성동



프로젝트 목표: 가상의 숙박업 비지니스를 창업한다는 가정하에 어떻게해야 비지니스를
성공적으로 이끌수 있을지 데이터 수집&분석을 통해 인사이트를 도출

관광+교통 데이터(로그) 분석하여 숙박업 비지니스에 유용한 인사이트를 도출
(숙박업 이용자들의 정보를 분석)

웹크롤링으로 숙박업소 리뷰, 가격 등 경쟁자들을 분석하고 비지니스 전략 수립

최종적으로 분석한 결과물을 웹에 구현하여 사용자들이 웹을 통하여 접근가능하도록 함
